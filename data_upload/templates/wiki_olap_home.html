Bem vindo ao WikiOlap!

O que é?

O WikiOlap é um repositório integrado que permite o armazenamento de diferentes conjuntos de dados. A partir do processamento desses dados nossa ferramenta permite o acesso aos dados integrados a partir de uma API REST. Com isso usuários podem colaborar enviando diferentes conjuntos de dados e acessando os conjuntos enviados por outros usuários. A partir desses dados integrados diferentes visualizações e análises podem ser feitas!
