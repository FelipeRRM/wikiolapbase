São 4 passos para enviar o seu conjunto de dados!

Siga abaixo e aprenda

  • 1º Passo

    Upload


    O primeiro passo para carregar seu conjunto de dados em nosso repositório é fazer o upload do seu arquivo! Para isso clique na opção "Enviar Dataset" no menu ao lado. Na nova página, clique no botão "ADICIONAR DATASET". Uma janela irá se abrir e você poderá selecionar seu dataset entre os arquivos de seu dispositivo. Selecione o arquivo e clique em "Iniciar envio". Pronto! Agora você pode passar para o 2º passo.


    Passo 2

  • 2º Passo

    Descrição


    No segundo passo você fornece algumas informações sobre o dataset que você selecionou no 1º passo. Todas essas informações devem ser preenchidas: título, descrição, fonte e email para contato. Essas informações facilitam para outros usuários encontrarem seu dataset caso desejam utilizá-lo. Após preencher essas informações você pode passar para o 3º passo.


    Passo 1 Passo 3

  • 3º Passo

    Tags


    No terceiro passo você tem a opção de adicionar tags as colunas do seu dataset! Por exemplo, suponha que você tem uma coluna cujos dados são valores de arrecadação estadual de IPVA, você poderia adicionar a tag "imposto " nessa coluna. Dessa forma quando você buscasse o termo "imposto " em nosso repositório o seu dataset teria maior probabilidade de aparecer! Além disso, isso facilita o cruzamento de dados similares dentro do repositório. Nesse passo você também pode renomear as colunas do seu dataset para nomes mais amigáveis se assim desejar, basta clicar sobre o nome da coluna e modificá-lo. Com isso pronto você pode passar para o 4º e último passo.

    Hierarquias


    Dados hierárquicos são estruturas de dados organizados no formato de árvore. Cada nó possui apenas um filho. Esse tipo de estrutura permite diferentes alternativas quando visualizações de dados são geradas. Nesse passo você pode definir hierarquias de dados em seu dataset.
